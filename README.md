## Running the website
```
    - cd ./ethicalcode

    - npm start

    - http://localhost:3000
```

## Running Docker
```
    - docker build -t ethicalcode-vscode -f Docker\.                
    devcontainer\Dockerfile Docker\.devcontainer

    - docker run -p 8080:8080 ethicalcode-vscode

    - http://localhost:8080
```
